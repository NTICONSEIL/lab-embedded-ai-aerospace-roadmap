# Embedded AI & Aerospace Systems Roadmap

## Overview

This repository documents a complete learning journey towards expertise in:

- Embedded Signal Processing
- Sensor Fusion
- State Estimation
- Autonomous Navigation
- Robotics Perception
- Embedded Artificial Intelligence
- Aerospace Embedded Systems
- Digital Twins

The objective is to progressively acquire the theoretical foundations and practical skills required for modern aerospace and autonomous systems applications.

The repository combines:

- Theoretical notes
- Mathematical foundations
- Python simulations
- Jupyter notebooks
- Practical projects
- Embedded implementations
- Aerospace-oriented case studies

---

## Motivation

Modern aerospace systems rely increasingly on intelligent embedded systems capable of:

- Processing sensor data in real time
- Estimating system states
- Performing autonomous navigation
- Detecting anomalies
- Executing AI algorithms on constrained hardware

Applications include:

- Drones
- UAVs
- Autonomous vehicles
- Satellites
- Robotics
- Industrial IoT systems

This repository serves both as:

- A personal competency development roadmap
- A teaching resource
- A technical portfolio
